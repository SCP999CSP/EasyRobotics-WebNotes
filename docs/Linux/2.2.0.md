# 2.2 File and Directory Operations (Command Line)

Working with files and directories is one of the most fundamental tasks in Linux.  
This section focuses on using the command line to **navigate the file system and manage files confidently**.

---

## Learning Goal

After completing this section, you should be able to:

- Navigate the Linux file system from the terminal
- Understand how directories and files are organized
- Perform common file and directory operations safely
- Locate files and understand disk usage information

---

## Why the Command Line Matters

Although graphical tools exist, the command line:
- Provides precise control
- Works consistently across systems
- Is essential for servers and remote environments

Mastering these operations builds a strong foundation for all advanced Linux tasks.
