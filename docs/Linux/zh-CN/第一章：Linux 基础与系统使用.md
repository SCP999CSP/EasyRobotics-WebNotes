# 第一章：Linux 基础与系统使用

本章将带你建立对 Linux 的**整体认知**，并掌握在终端环境下的**基本操作能力**。  
目标不是“背命令”，而是理解 **Linux 是如何被使用的**，以及如何通过命令行高效完成日常任务。

---

## 本章学习目标

完成本章后，你将能够：

- 理解 Linux 在开发与服务器中的角色
- 熟悉 Linux 的基本使用环境（终端 / Shell）
- 使用常见命令完成文件与目录管理
- 查看和处理文本文件内容
- 使用 vim 编辑器进行基础文本编辑

---

## 本章内容结构

### 1. Linux 基础认知
- Linux 是什么
- Linux 的应用场景

- 终端与 Shell 的基本概念
- 文件与目录的常用操作指令
- 文件内容查看与简单文本处理
- vim 编辑器的基础使用

## 学习建议

- 本章以**动手实践**为主，建议边看边在终端中操作
- 不要求一次记住所有命令，重点是理解使用场景
- 后续章节会在此基础上逐步引入权限、进程、网络等内容
